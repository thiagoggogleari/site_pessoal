Portfólio

Há no mercado uma placa de desenvolvimento chamada ESP32 e com ela é possível realizar protótipos. Porém ela vem com pinos que não são muito apropriados manter fios conectados, pois podem facilmente se soltar ao manuseá-la.

Neste projeto desenvolvi uma placa expansora aonde os bornes com parafusos, estão insterligados aos pinos da placa Esp32. Este conjunto utilizei na parte de automação da minha residência, aonde eu conecto lâmpadas, sensores de presença, sensor de gás e sensores de temperatura.

Para o jogo Minecraft há disponível um software Open Source para fazer modificações no jogo, o MCreator. Entre os mods há o "Minecraft Link",ele permite a comunicação do jogo com a placa Arduino. Sendo possível o acionamento de leds através do jogo.

Em conjunto com o Arduino conectei uma placa Esp8266 que tem comunicação via Wi-fi e através dela enviar comandos para interagir com as lâmpadas conectadas à placa Esp32 da automação da minha residência.

Neste projeto é enviado mensagens para um bot no Telegram e as mensagens aparecem em um display. O display está conectado à uma placa Esp8266 que se comunica via MQTT à uma instância Node-Red rodando na IBM Cloud (serviço gratuito). O Node-Red é responsável por receber as mensagens do bot e a comunicação com o Esp8266 é feita via protocolo MQTT.

Também possuo um repositório no GitHub aonde estou colocando códigos que venho desenvolvendo nos meus estudos.
